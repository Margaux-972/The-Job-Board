# The Job Board

Ceci est un exercice pratique afin d'assimiler les bases de l'initialisation du projet React.
Comme son nom l'indique, le rendu de cet exercice est un tableau recensant diverses offres d'emploi.
Il n'y a pas de backend ni de db, le but étant juste un display.

## Hébergement Netlify

Cet exercice me permet de m'entraîner sur l'hébergement de site avec Netlify.
